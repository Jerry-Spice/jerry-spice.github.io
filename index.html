Current Project - C Libraries

I am currently learning C for a class that I am in and I've been having trouble doing many simple operations like reading files, dealing with strings, and slicing data how I would like to. As a result I've been using some of my spare time to develop some libraries for myself in order to make life easier. I'm most familiar with Python so I've been modeling my libraries based on the functionality of Python libraries and how strings and file reading works in that language. I've also been reading some about using special structures to make life easier so I've tried one of those so far.

Strings.h

I've found that dealing with strings in C is particularly frustrating. Especially dealing with variable size strings. As a result I set out to create a simple string structure. The goal is to resize and reallocate memory dynamically because I don't like seeing all the memory allocation and math in my main function. As a result I've developed the following structure and functions:

struct String { char* content; int length; int size; }; typedef struct String String;
  • String* createString(int size)
  • void fillString(String* target, char* content)
  • void printString(String* target)
  • String* substring(String* target, int start, int end)
  • int countOcurrancesInString(String* target, String* pattern)
  • int* getOcurrancesInString(String* target, String* pattern)
  • int splitString(String* target, String* delimiter, String** stringSplits)
  • int isEqual(String* string1, String* string2)
